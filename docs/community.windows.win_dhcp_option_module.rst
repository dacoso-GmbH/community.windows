*********************************
community.windows.win_dhcp_option
*********************************

**Manage Windows Server DHCP Options**



.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Manage Windows Server DHCP Options (IPv4/IPv6)
- Adds, Removes and Modifies DHCP Options
- Task should be delegated to a Windows DHCP Server



Requirements
------------
The below requirements are needed on the host that executes this module.

- This module requires Windows Server 2012 or Newer

Examples
--------

.. code-block:: yaml

    win_dhcp_option:
              state: present
              version: "IPv6"
              type: "server"
              optionid: 24
              value: "server.ipv6.intern.systemuser.de"

          win_dhcp_option:
              state: present
              version: "IPv6"
              type: "scope"
              optionid: 24
              value: "scope.ipv6.intern.systemuser.de"
              prefix: "2001:5cc1::"
        
          win_dhcp_option:
              state: present
              version: "IPv4"
              type: "reservation"
              optionid: 15
              value: "reserved.ipv4.intern.systemuser.de"
              reservedip: "192.168.222.222"
